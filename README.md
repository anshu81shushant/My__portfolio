# My Portfolio

Personal one-page website built with HTML, CSS, and vanilla JavaScript.

### Features
- Light / Dark mode toggle  
- Typing animation in header  
- Popup contact form  
- Scroll-to-top button  
- Responsive (mobile-friendly)  

### Technologies
- HTML5  
- CSS3  
- Vanilla JavaScript  

### How to run

# Clone repo
git clone https://github.com/anshu81shushant/my-portfolio.git

# Go to folder
cd my-portfolio

# Start simple server
python -m http.server 8000
